# jka-multihack

This is a multihack for the idTech3 based game Jedi Academy with these features:

- Wallhack
- Player glow (team and duel aware)
- Cheat unlocker
- Triggerbot
- Aimbot
- Anti Grip
- Anti Mind Trick

For a detailed writeup check out [this blog post](https://bananamafia.dev/post/multihack/).
